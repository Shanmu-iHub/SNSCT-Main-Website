# SNS College of Technology (SNSCT) - Main Website

![SNS College of Technology](./Assets/SNSCT%20Faviocn.png)

Welcome to the official repository for the **SNS College of Technology** landing page and main website. SNSCT is India's First Design Thinking Institution, located in Coimbatore, Tamil Nadu, merging world-class technical excellence with an innovative engineering mindset.

## 🌟 Key Highlights

- **Design Thinking Framework:** A patented, GenAI-powered approach focusing on empathy and creative problem solving.
- **5 Pillars of Development:** Learning, Upskilling, Innovation, Networking, and Character Building.
- **SNS iHub:** State-of-the-art incubation and innovation center (India's Y-Combinator equivalent).
- **World-Class Facilities:** Includes the SPINE Center (swimming pool, gyms, dance studios, etc.), modern labs (Aerospace, IoT, EV, VLSI), and premium infrastructure.
- **Extensive Alumni Network:** A robust community with graduates placed in top-tier product and tech companies like Google, Amazon, and successful startup founders.

## 📂 Project Structure

This static website project includes multiple pages and asset directories:
- `index.html` - The main landing page.
- `about.html` - Information about the institution, vision, and mission.
- `facilities.html` / `gallery.html` - Showcases campus infrastructure, labs, and amenities.
- `events.html` - Highlighting technical symposiums, cultural events, and sports meets.
- `sports.html` / `innovation.html` - Details about the Beyond Programs initiatives.
- `success.html` - Alumni success stories and career outcomes.
- `Assets/` - All site imagery, including high-res lab photos, event flyers, and logos.

## 🛠️ Technology Stack

- **HTML5:** Semantic structure and page layouts.
- **Tailwind CSS:** Modern, utility-first styling for responsive design, rich gradients, and dynamic hover effects.
- **Vanilla JavaScript:** For interactive components like mobile navigation, chat bubbles, and dynamic carousels.

## 🚀 Getting Started

To view or modify the website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Shanmu-iHub/SNSCT-Main-Website.git
   ```
2. Open the project folder.
3. Use a local server (like VS Code's Live Server extension) to run the site:
   - Right-click `index.html` and select **"Open with Live Server"**.
